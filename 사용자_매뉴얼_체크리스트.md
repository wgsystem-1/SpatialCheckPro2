# Spatial Check Pro 검수 체크리스트

검수를 시작하기 전에 확인해야 할 사항들을 체크리스트로 정리했습니다.

---

## 검수 전 체크리스트

### ✅ 시스템 준비

- [ ] Windows 10 이상 (64비트) 확인
- [ ] .NET 8.0 Runtime 설치 확인
- [ ] 최소 4GB RAM 확인 (권장: 8GB 이상)
- [ ] 디스크 공간 충분한지 확인 (최소 500MB + 검수 결과 저장 공간)

### ✅ 프로그램 준비

- [ ] 프로그램 실행 파일 위치 확인
- [ ] Config 폴더에 검수 규칙 CSV 파일 존재 확인
  - [ ] `1_table_check.csv`
  - [ ] `2_schema_check.csv`
  - [ ] `3_geometry_check.csv`
  - [ ] `4_attribute_check.csv`
  - [ ] `5_relation_check.csv`
  - [ ] `codelist.csv`
  - [ ] `geometry_criteria.csv`
- [ ] 프로그램 정상 실행 확인

### ✅ 데이터 준비

- [ ] 검수할 FileGDB 파일 위치 확인
- [ ] FileGDB 파일이 다른 프로그램에서 열려있지 않은지 확인
- [ ] FileGDB 파일 읽기 권한 확인
- [ ] FileGDB 파일 경로에 한글/특수문자 없는지 확인
- [ ] FileGDB 파일이 손상되지 않았는지 확인

### ✅ 검수 계획

- [ ] 검수할 단계 결정 (1~5단계)
- [ ] 각 단계별 검수 규칙 확인
- [ ] 검수 옵션 설정 계획 (병렬 처리, 스트리밍 모드 등)
- [ ] 검수 결과 저장 위치 결정

## 검수 중 체크리스트

### ✅ 검수 실행

- [ ] 폴더 선택 완료
- [ ] 검수 설정 완료
- [ ] 검수 시작 버튼 클릭
- [ ] 검수 진행 화면에서 진행률 확인
- [ ] 예상 소요 시간 확인

### ✅ 진행 상황 모니터링

- [ ] 전체 진행률 확인
- [ ] 단계별 진행률 확인
- [ ] 현재 작업 항목 확인
- [ ] 오류 발생 시 로그 확인
- [ ] 메모리 사용량 모니터링 (필요 시)

---

## 검수 후 체크리스트

### ✅ 결과 확인

- [ ] 검수 완료 메시지 확인
- [ ] 대시보드에서 전체 요약 확인
- [ ] 단계별 상세 결과 확인
- [ ] 오류 목록 확인
- [ ] 통계 차트 확인

### ✅ 결과 저장

- [ ] 검수 결과 데이터베이스 저장 확인 (`ValidationResults.db`)
- [ ] 오류 Shapefile 내보내기 (필요 시)
- [ ] 검수 보고서 생성 (PDF/HTML)
- [ ] 로그 파일 확인 (문제 발생 시)

### ✅ 결과 분석

- [ ] 심각도별 오류 분류 (CRITICAL → MAJOR → MINOR → INFO)
- [ ] 오류 유형별 집계
- [ ] 가장 많이 발생한 오류 확인
- [ ] 오류 위치 확인 (GIS 소프트웨어 사용)

### ✅ 후속 작업

- [ ] 오류 수정 계획 수정
- [ ] 수정된 데이터 재검수 계획
- [ ] 검수 결과 문서화
- [ ] 검수 보고서 공유 (필요 시)

---

## 문제 발생 시 체크리스트

### ✅ 오류 발생 시

- [ ] 오류 메시지 확인
- [ ] 로그 파일 확인 (`Logs/app-YYYY-MM-DD.log`)
- [ ] 사용자 매뉴얼의 "문제 해결" 섹션 참조
- [ ] 오류 코드 확인 (오류 코드 참조표 참조)
- [ ] 기술 지원 문의 (필요 시)

### ✅ 성능 문제 시

- [ ] 메모리 사용량 확인
- [ ] 디스크 공간 확인
- [ ] 다른 프로그램 종료
- [ ] 병렬 처리 옵션 확인
- [ ] 스트리밍 모드 활성화 확인
