## 문제 해결

### ✅ 아이콘이 표시되지 않는 경우

- [ ] 파일 경로 확인
- [ ] 프로젝트 파일에 파일 참조 확인
- [ ] 빌드 출력 폴더에 파일 존재 확인
- [ ] 리소스 경로 형식 확인 (`pack://application:,,,/`)

### ✅ 이미지가 표시되지 않는 경우

- [ ] 이미지 파일이 리소스로 포함되었는지 확인
- [ ] XAML에서 `Source` 속성 경로 확인
- [ ] 이미지 파일 형식 확인 (PNG 권장)
- [ ] 파일 크기 확인 (너무 크면 로딩 지연 가능)
