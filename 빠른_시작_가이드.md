# Spatial Check Pro 빠른 시작 가이드

**5분 안에 검수 시작하기**

---

## 1단계: 프로그램 실행 (30초)

1. `SpatialCheckPro.GUI.exe` 더블 클릭
2. 프로그램이 시작될 때까지 대기

> ✅ **확인**: 왼쪽 사이드바에 메뉴가 표시되면 준비 완료

---

## 2단계: 폴더 선택 (1분)

1. 사이드바에서 **"폴더 선택"** 클릭
2. 검수할 FileGDB가 있는 폴더 선택
3. 폴더 내 .gdb 파일 목록 확인
4. **"확인"** 버튼 클릭

> ✅ **확인**: 선택한 폴더 경로가 화면에 표시되면 성공

---

## 3단계: 검수 설정 (2분)

1. 사이드바에서 **"검수 진행"** 클릭
2. 검수할 단계 선택 (기본: 모두 선택)
   - ✅ 1단계: 테이블 검수
   - ✅ 2단계: 스키마 검수
   - ✅ 3단계: 지오메트리 검수
   - ✅ 4단계: 속성 관계 검수
   - ✅ 5단계: 공간 관계 검수
3. **"검수 진행"** 버튼 클릭

> ✅ **확인**: 검수 진행 화면이 표시되면 시작됨

---

## 4단계: 검수 완료 대기 (시간 가변)

- 검수 진행 화면에서 진행률 확인
- 완료될 때까지 대기
- 필요 시 **"취소"** 버튼으로 중단 가능

> ✅ **확인**: "검수 완료" 메시지가 표시되면 성공

---

## 5단계: 결과 확인 (1분)

1. 사이드바에서 **"검수 결과"** 클릭
2. 대시보드에서 전체 요약 확인
3. 단계별 탭에서 상세 결과 확인
4. 오류 목록 확인

> ✅ **확인**: 오류 수와 통계가 표시되면 성공

---

## 문제 발생 시

1. **프로그램이 실행되지 않음**
   - .NET 8.0 Runtime 설치 확인
   - 관리자 권한으로 실행

2. **파일을 읽을 수 없음**
   - 파일 경로에 한글/특수문자 확인
   - 파일이 다른 프로그램에서 열려있는지 확인

3. **검수가 너무 느림**
   - 병렬 처리 옵션 활성화
   - 불필요한 검수 규칙 비활성화
